# Validação de Produção (smoke test manual)

Checklist de smoke test para confirmar que Produção continua íntegra.

Este documento referencia o checklist detalhado em `docs/production-audit-checklist.md`.

## Smoke (rápido)

1. Criar OP com itens.
2. Abrir Controle de Produção e confirmar itens no setor inicial.
3. Mover item para o próximo setor e confirmar movimentação criada.
4. Abrir Dashboard TV e confirmar atualização.
5. Concluir item no último setor.
6. Testar multi-tenant (A não enxerga B).
